## xPDO::getDebug

Возвращает состояние отладки для соединения XPDO.

## Синтаксис

API Docs: <https://api.modx.com/revolution/2.2/db_core_xpdo_xpdo.class.html#\xPDO::getDebug()>

```php
boolean getDebug ()
```

## Пример

Получите, включен ли режим отладки или нет.

```php
$isDebugOn = $xpdo->getDebug();
```
