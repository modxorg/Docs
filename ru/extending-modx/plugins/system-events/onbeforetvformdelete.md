## Событие: OnBeforeTVFormDelete

Срабатывает перед удалением TVа в менеджере.

Служба: 1 - Parser Service Events
Group: Template Variables

## Параметры события

| Имя | Описание                         |
| --- | -------------------------------- |
| tv  | Ссылка на объект modTemplateVar. |
| id  | Идентификатор TV.                |
