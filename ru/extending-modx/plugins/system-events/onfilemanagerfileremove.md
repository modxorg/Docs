## Событие: OnFileManagerFileRemove

 Срабатывает после удаления файла через менеджер.

 Служба: 1 - Parser Service Events
 Группа: Нет

## Параметры события

 | Имя    | Описание                                          |
 | ------ | ------------------------------------------------- |
 | path   | Содержит полный путь к файлу, который был удален. |
 | source | Содержит объект медиаресурса.                     |
