## Событие: OnManagerLoginFormRender

Запускается, когда форма входа отображается для менеджера MODX. Полезно для вставки пользовательского HTML в форму входа.

Служба: 2 - Manager Access Events
Группа: Нет

## Параметры события

Нет.
