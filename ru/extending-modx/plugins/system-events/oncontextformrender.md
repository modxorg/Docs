## Событие: OnContextFormRender

Запускается после загрузки формы редактирования контекста.

Служба: 2 - Manager Access Events
Группа: modContext

## Параметры события

| Имя     | Описание                                         |
| ------- | ------------------------------------------------ |
| key     | Ключ контекста.                                  |
| context | Ссылка на объект modContext.                     |
| mode    | Либо 'upd' или 'new', в зависимости от ситуации. |
