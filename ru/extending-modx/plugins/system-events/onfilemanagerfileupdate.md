## Событие: OnFileManagerFileUpdate

Запускается после обновления файла через менеджер.

Служба: 1 - Parser Service Events
Группа: Нет

## Параметры события

| Имя    | Описание                                   |
| ------ | ------------------------------------------ |
| path   | Содержит полный путь к обновленному файлу. |
| source | Содержит объект медиаресурса.              |
