## Событие: OnBeforeUserFormDelete

Запускается до того, как пользователь будет удален в менеджере.

Служба: 1 - Parser Service Events
Group: Users

## Параметры события

| Имя  | Описание                    |
| ---- | --------------------------- |
| user | Ссылка на объект modUser.   |
| id   | Идентификатор пользователя. |
