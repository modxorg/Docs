## Событие: OnPluginRemove

Срабатывает сразу после удаления плагина из базы данных.

Служба: 2 - Manager Access Events
Группа: modPlugin

## Параметры события

| Имя    | Описание                    |
| ------ | --------------------------- |
| plugin | Ссылка на объект modPlugin. |
