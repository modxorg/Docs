## Событие: OnBeforeTempFormDelete

Запускается до удаления шаблона в менеджере.

Служба: 1 - Parser Service Events
Group: Templates

## Параметры события

| Имя      | Описание                      |
| -------- | ----------------------------- |
| template | Ссылка на объект modTemplate. |
| id       | Идентификатор шаблона.        |
