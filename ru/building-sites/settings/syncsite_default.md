-   **Имя**: Очищать кэш по умолчанию
-   **Тип**: Да/Нет  
-   **По умолчанию**: Да
-   **Доступен в**: Revolution 2.3.3+  

Если выбрано «Да», при сохранении Ресурса кэш будет очищаться по умолчанию.
