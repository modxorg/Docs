## auto\_check\_pkg\_updates

**Имя**: Automatic Check for Package Updates
**Тип**: Да/Нет
**По умолчанию**: Да

Если «Да», MODX автоматически проверит наличие обновлений для пакетов в управлении пакетами. Это может замедлить загрузку списка пакетов.
