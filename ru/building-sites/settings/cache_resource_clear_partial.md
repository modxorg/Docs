-   **Имя**: Частичная очистка кэша для указанных Контекстов
-   **Тип**: Да/Нет  
-   **По умолчанию**: Нет
-   **Доступно в**: Revolution 2.7.0+

Когда включено, MODX обновит кэш Ресурсов только для указанных Контекстов.
