-   **Имя**: Максимальное время жизни сборщика мусора для сессии   
-   **Тип**: String  
-   **По умолчанию**: 604800  
-   **Доступен в**: Revolution 2.1.1+

Позволяет настраивать ini-параметр PHP `session.gc_maxlifetime` при использовании `modSessionHandler`.
