-   **Имя**: Включение блокировки JS/CSS файлов кеша Менеджера 
-   **Тип**: Да/Нет  
-   **По умолчанию**: Да  
-   **Доступен в**: Revolution 2.2.0+

Блокировка файла кеша. Установите значение "Нет", если файловая система - NFS.
