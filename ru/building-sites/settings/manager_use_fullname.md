-   **Имя**: Отображать «Полное имя» в шапке Системы Управления
-   **Тип**: Да/Нет  
-   **По умолчанию**: Нет
-   **Доступен в**: Revolution 2.4.0+  

Если параметр включен, в шапке Системы Управления будет отображаться содержимое поля «Полное имя» вместо «Имя пользователя»
