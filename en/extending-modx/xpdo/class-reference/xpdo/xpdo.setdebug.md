## xPDO::setDebug

Sets the debug state for the XPDO connection.

## Syntax

API Docs: <https://api.modx.com/revolution/2.2/db_core_xpdo_xpdo.class.html#\xPDO::setDebug()>

``` php
void setDebug ([boolean $v = true])
```

## Example

``` php
$xpdo->setDebug(true);
```
