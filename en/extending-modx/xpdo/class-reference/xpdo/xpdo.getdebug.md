## xPDO::getDebug

Returns the debug state for the XPDO connection.

## Syntax

API Docs: <https://api.modx.com/revolution/2.2/db_core_xpdo_xpdo.class.html#\xPDO::getDebug()>

``` php
boolean getDebug ()
```

## Example

Get whether debug mode is on or not.

``` php
$isDebugOn = $xpdo->getDebug();
```
