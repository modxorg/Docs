## Event: OnCategoryBeforeRemove

Happens right before a category is removed.

Service: 2 - Manager Access Events
Group: Categories

## Event Parameters

| Name     | Description                            |
| -------- | -------------------------------------- |
| category | A reference to the modCategory object. |
