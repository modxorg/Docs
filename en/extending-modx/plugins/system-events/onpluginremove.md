## Event: OnPluginRemove

Fires just after the plugin is removed from the database

Service: 2 - Manager Access Events
Group: modPlugin

## Event Parameters

| Name   | Description                          |
| ------ | ------------------------------------ |
| plugin | A reference to the modPlugin object. |
