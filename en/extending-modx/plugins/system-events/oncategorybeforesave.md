## Event: OnCategoryBeforeSave

Happens right before a category is saved.

Service: 2 - Manager Access Events 
Group: Categories

## Event Parameters

| Name     | Description                            |
| -------- | -------------------------------------- |
| id       | The ID of the modCategory object.      |
| category | A reference to the modCategory object. |
