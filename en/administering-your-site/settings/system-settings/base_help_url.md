base\_help\_url
---------------

**Name**: Base Help URL   
**Type**: Textfield   
**Default**: <http://rtfm.modx.com/display/revolution20/>  
**Available In**: Revolution 2.1+

The base URL by which to build the Help links in the top right of pages in the manager.
