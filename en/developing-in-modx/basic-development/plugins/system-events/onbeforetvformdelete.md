## Event: OnBeforeTVFormDelete

Fires before a TV is deleted in the manager.

Service: 1 - Parser Service Events 
Group: Template Variables

## Event Parameters

| Name | Description                               |
| ---- | ----------------------------------------- |
| tv   | A reference to the modTemplateVar object. |
| id   | The ID of the TV.                         |
